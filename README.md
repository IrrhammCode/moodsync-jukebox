# MoodSync Jukebox

A real-time, shared music listening experience powered by face-api.js, Socket.io, Turbopuffer, and ElevenLabs.

## Getting Started

1. Set up your `.env` file based on `.env.example`.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the dev servers (both frontend and backend concurrently):
   ```bash
   npm run dev
   ```
   - Frontend runs on: `http://localhost:3000`
   - Backend Socket.io runs on: `http://localhost:3001`
